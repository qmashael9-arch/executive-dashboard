# Aseer Executive Dashboard

منصة أداء تنفيذية تفاعلية لتجمع عسير الصحي.

## النشر على GitHub Pages
ارفع جميع الملفات والمجلدات كما هي إلى جذر المستودع، ثم فعّل GitHub Pages من الفرع `main` والمجلد `/root`.

## التعديل
- فعّل وضع التعديل من أعلى الصفحة.
- استخدم أزرار القلم لتعديل المؤشرات والمستشفيات والملاحظات.
- احفظ التعديلات محليًا أو صدّر نسخة JSON.
